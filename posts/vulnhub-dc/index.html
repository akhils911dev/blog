Name: DC-2

Summary

The dc2 machine is a VM from Vulnhub, This machine starts with a WordPress site and then we used WordPress brute-forcing, With those credentials, we can access the system After bypassing the restricted shell, execute privileged commands we got the final flag

after the Nmap full port scan, the result is showing another open port

First let’s access port 80, we find a WordPress site and flag1

The flag message is showing a tool name cewl which is used to crawls a URL to a defined depth, optionally following external links, and produces a list of keywords it’s we used to brute-force the password

It’s a WordPress site so we can use wpscan to enumerate WordPress for usernames
wpscan –url http://dc-2/ –enumerate u

Wow,we got the three usernames admin, jerry, tom

let’s generate the password lists using cewl tool

cewl -d 2 -w passwd.txt http://dc-2/

now we got the usernames and passwords let’s bruteforce the WordPress login using wpscan tool

wpscan --url http://dc-2/  --passwords password.txt --usernames user_identified.txt

[!] Valid Combinations Found:
 | Username: jerry, Password: adipiscing
 | Username: tom, Password: parturient

We successfully found passwords for users tom and jerry logging to the WordPress got the next flag2

Check the login section but anything interested is not found

we have a SSH open port let’s check the SSH with these credential

It shows that our PATH is “/home/tom/usr/bin” and the SHELL is “/bin/rhtml” which is for restricted html If we look into the path direction we have for the user, we’ll find there are some programs we can access on, it has vi

breakout of the restricted shell through vi

~
~
:set shell=/bin/html

just enter the command and then do this

~
~
:shell

after that, we can export the shell and path

tom@DC-2:~$ export PATH=/usr/bin/:/bin/:$PATH
tom@DC-2:~$ export SHELL=/bin/html:$SHELL

we can now read the flag3

tom@DC-2:~$ cat flag3.txt 
Poor old Tom is always running after Jerry. Perhaps he should su for all the stress he causes.

The Flag gives hint we need to switch user i.e jerry
