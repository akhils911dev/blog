Summary

The matrix VM is a virtual machine from vulnhub it’s a boot2root machine first, we need to find the IP and then scan the IP it discovered some open ports and services after checking those services we got a hint from one web server, with that hint we can access another web server, we got an encoded string from these web server source page decode the strings it’s a bash command with a file name, access the file from the webserver it has an another encode data that includes username and password. With those credentials, we can access the system through ssh. After bypassing a restricted shell, execute the privileged command then we got a flag

Privilege escalation

Once here I try to use some commands but bad luck don’t have access to use any command because it’s a restricted shell, this is commonly used for restricting the users to certain actions, in this case, i’ll need to find a way to bypass the shell

guest@porteus:~$ ls
-rbash: /bin/ls: restricted: cannot specify `/' in command names
guest@porteus:~$ whoami
-rbash: whoami: command not found
guest@porteus:~$ echo $SHELL
/bin/rbash
guest@porteus:~$ echo $PATH
/home/guest/prog
guest@porteus:~$ echo /home/guest/prog/*
/home/guest/prog/vi
guest@porteus:~$

It shows that our PATH is “/home/guest/prog” and the SHELL is “/bin/rbash” which is for restricted bash. If we look into the path direction we have for the user, we’ll find there is only one program we can access which is vi
